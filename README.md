# CineNow-Movies
Thrilled to share a responsive Movie Website built using HTML, CSS, and JavaScript. The project provides an interactive interface for browsing movies, viewing posters and details, and searching for favorite titles. It features a clean design and dynamic content display, demonstrating modern front-end web development skills. 
A modern and responsive movie browsing website developed using HTML, CSS, and JavaScript. This project allows users to explore movies through an attractive interface featuring movie posters, titles, ratings, and descriptions. The website includes search functionality, smooth navigation, and a mobile-friendly design to enhance user experience. It demonstrates core front-end development concepts such as responsive layouts, DOM manipulation, and interactive user interfaces. This project was created as part of an academic web development project to showcase practical skills in building dynamic and visually appealing websites.
